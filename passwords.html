Password Security

Most websites require a person to sign up to an account. To create an account a password is always required. This password allows only the specified user to access the account.


Remember to use the following when creating a password;

Numbers

Avoid using the default passwords such as ‘1234’. Use a range of numbers (but not significant dates. I.E birthday.)

Special Characters

If allowed use a special character such as '!'. Use plenty of characters. A long password is more secure than a short password.

Uniqueness

A password should be unique for each account. It should not be shared over multiple sites. Avoid using common words or multiple number/ letters. E.g., Balloon222 or 123456

Why?


The best way to stay safe online is by creating a password that is secure.


Having a secure password is essential for staying safe online and keeping personal information protected.

Hackers who intend on using your accounts to commit crimes will try and obtain your password to gain access. About 80% of data breaches in 2019 were caused by password compromise. If they succeed this could then lead on to crimes such as identity theft. The hacker could also use your accounts to gain paymeny information from you. Then commit further crimes with your details.


It can be tempting to create a password using memorable information. Like personal details. Although easy to remember these words/ numbers make it easy for hackers to correctly guess your password and access your accounts.
